## VS Code Computer Use for Windows

### Editing and navigation

- Для code-heavy задач предпочитай shell/files/editor-native flows; Computer Use здесь нужен только для GUI-specific действий.
- Не используй Computer Use для automation самого Codex UI или terminal shell workflow, если есть прямой shell/tool path.

### Safety

- Не подтверждай destructive source control actions без явного намерения пользователя.
