## File Explorer Computer Use for Windows

### Navigation

- Для переходов по списку и панели адреса предпочитай `get_app_state -> click -> get_app_state`.
- Если shell reused existing window or tab, не делай опасные cleanup actions без отдельного ownership proof.

### Safety

- Не перемещай, не удаляй и не переименовывай файлы без явного намерения пользователя.
