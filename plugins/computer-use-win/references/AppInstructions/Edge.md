## Edge Computer Use for Windows

### Browser-first rule

- Если задача чисто web-native, сначала предпочитай Browser/IAB path, а не desktop Computer Use.
- К desktop Computer Use переходи, когда нужен именно Windows app workflow, browser chrome или native dialogs.

### Safety

- Не выполняй отправку форм, платежи и входы без явного намерения пользователя и подтверждения risky action.
