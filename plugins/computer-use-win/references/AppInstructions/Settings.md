## Settings Computer Use for Windows

### Navigation

- Settings часто содержит externally visible и privacy-sensitive actions.
- После клика по важным toggles или navigation entries делай новый `get_app_state`.

### Safety

- Изменения privacy, security, accounts и system-wide settings считать risky.
- Не подтверждай destructive or externally visible settings changes без явного подтверждения пользователя.
