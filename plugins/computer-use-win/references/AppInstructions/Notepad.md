## Notepad Computer Use for Windows

### Editing text

- Сначала получи `get_app_state`, затем кликни по текстовой области и только после этого выполняй ввод.
- После клика на текстовую область повторный `get_app_state` или verify-step полезнее, чем слепая серия действий.

### Safety

- Не закрывай окно и не подтверждай сохранение без явного намерения пользователя.
- Если текст уже есть, не перезаписывай его молча.
